# react-native

## create project
```javascript
npm uninstall -g react-native-cli @react-native-community/cli

npx react-native@X.XX.X init AwesomeProject --version X.XX.X
```

## run project
```javascript
npx react-native start

[//]: # (r - reload the app)

[//]: # (d - open developer menu)

[//]: # (i - run on iOS)

[//]: # (a - run on Android)

npx react-native run-android
```
